
# preprocesamiento-cienciadatos

##  Objetivo
Este proyecto tiene como objetivo aplicar técnicas de preprocesamiento de datos con la finalidad de preparar conjuntos de datos antes de su análisis o modelado en ciencia de datos.  
En donde se incluye pasos como limpieza, transformación, normalización y manejo de valores faltantes.

##  Estructura básica de carpetas
preprocesamiento-cienciadatos/


Data/  Conjunto de datos brutos y procesados

Notebooks/  Notebooks de Jupyter para análisis y pruebas

Scripts/  Scripts en Python para limpieza y transformación

Docs/  Documentación y reportes

Results/  Resultados o salidas del preprocesamiento
